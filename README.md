# Spotify Clone - Music Streaming Platform

A full-stack music streaming application inspired by Spotify, built with React, Node.js, Express, and MySQL. Users can create accounts, log in, browse songs, listen to music, manage playlists, and receive personalized recommendations.

## Features

### 🎵 Music Player
- **Play/Pause/Next/Previous**: Standard playback controls
- **Volume Control**: Adjust audio volume
- **Progress Bar**: Seek through songs
- **Queue Management**: View and manage upcoming songs

### 📚 Library Management
- **My Music**: Access all saved songs and albums
- **Playlists**: Create, edit, and delete custom playlists
- **Liked Songs**: Automatically populated with liked tracks
- **Recently Played**: Track listening history

### 👥 User Features
- **Authentication**: Secure registration and login
- **User Profile**: View profile information
- **Following**: Follow artists and users
- **Comments**: Leave comments on songs and albums
- **Recommendations**: Personalized song suggestions

### 🌐 Discovery
- **Search**: Find songs, artists, and albums
- **Browse**: Explore curated playlists and charts
- **Artist Pages**: View artist discography and details
- **Album Pages**: Browse album tracks and information
